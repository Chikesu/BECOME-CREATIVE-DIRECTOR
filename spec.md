\# AI Video Production Agent



\## User

Marketing team



\## Problem

Video 30s mất 1–2 ngày sản xuất



\## Solution

Agent tự generate script, storyboard, shot list, prompt video



\## Input

Product info + target audience



\## Output

3 video concept + 1 video hoàn chỉnh



\## Workflow

1\. Phân tích brief

2\. Generate concept

3\. Viết script

4\. Tạo video

5\. Compile output



\## AI Tools

ChatGPT, Runway, ElevenLabs
